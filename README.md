# Boukedjar Ikram — Portfolio

## How to run
Open `index.html` directly, or open the folder in VS Code and click **Go Live**.

## Required profile picture
Put your profile photo here exactly:

```text
assets/profil.png
```

The portfolio is already coded to load that file in the hero/about section.

## Files
- `index.html` — page content
- `style.css` — full white/blue design + profile picture layout
- `script.js` — mobile menu, project filters, animations
- `assets/` — add your `profil.png` here

## Links to update
Search for `href="#"` in `index.html` and replace with your real GitHub, LinkedIn, Kaggle, and project demo links.
